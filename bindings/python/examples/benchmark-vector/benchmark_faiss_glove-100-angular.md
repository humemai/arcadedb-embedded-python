- Queries: 10,000
- Metric: Cosine

- **Name**: glove-100-angular

- **Metric**: Cosine
- **K Values**: [10, 50, 100]

## Targets & Use Cases

| k      | Recall target | Use case                   |
| ------ | ------------- | -------------------------- |
| 10     | 0.85–0.95     | Small corpora only         |
| **50** | **≥ 0.95**    | **Default RAG**            |
| 100    | 0.90–0.93     | Very large / noisy corpora |

As k increases, Recall@k can go down.

**Note:**
- **Metric Equations**:
  - **Euclidean**: Similarity = $1 / (1 + d^2)$ (Higher is better)
  - **Cosine**: Distance = $(1 - \cos(\theta)) / 2$ (Lower is better)
